# ⚽ GoalMaster - Landing Page Profesional

## 📌 Descripción

GoalMaster es una landing page profesional enfocada en el entrenamiento y análisis de fútbol mediante inteligencia artificial.
El objetivo es captar usuarios, generar interés y aumentar la conversión mediante una experiencia visual moderna e interactiva.

---

## 🚀 Tecnologías utilizadas

* HTML5 (estructura semántica)
* CSS3 (Flexbox, Grid, Variables)
* Diseño responsive (Mobile First)
* Accesibilidad (WCAG AA)
* Animaciones y transiciones CSS

---

## 🎮 Funcionalidades destacadas

* Diseño moderno y atractivo
* Secciones estructuradas (Hero, Beneficios, Features, Testimonios, CTA)
* Minijuego interactivo para mejorar la experiencia del usuario
* Navegación intuitiva
* Optimización para dispositivos móviles
